# Command-line-Browser
A browser that runs in your terminal and extracts text from the site you entered, with links highlighted.
Simply run the file and you will be prompted to enter your website. The program will then display the text
on the page the url leads to, highlighting the links in blue. The program will also save the contents of
the web page to a file and store it locally in its own directory.


REQUIREMENTS:
Please make sure you have the following python packages installed:
  -requests
  -BeautifulSoup (from bs4)
  -Colorama

Feel free to add suggestions if there's something I can do to improve this program.
